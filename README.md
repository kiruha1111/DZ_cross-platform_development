# Лабораторная работа #5 — Курьерская служба

## Описание проекта
Консольное приложение для управления курьерской службой с возможностью:
- Добавления обычных и авто-курьеров
- Сохранения/загрузки данных в XML
- Просмотра и управления списком курьеров

## Структура проекта
**Timirbaev_Erik_Lab_5/**
- `Program.cs` - Главная программа с меню
- `TimirbaevCourier.cs` - Базовый класс курьера
- `TimirbaevAutoCourier.cs` - Класс авто-курьера
- `TimirbaevPark.cs` - Управление штатом курьеров
- `TimirbaevUtilities.cs` - Вспомогательные методы
- `AssemblyInfo.cs` - Информация о сборке
- `App.config` - Конфигурация приложения
- `Проверка для Эрика` - check

## Основные классы

### 1. TimirbaevCourier
Базовый класс курьера с полями:
- `id` - уникальный идентификатор
- `firstname`, `surname` - имя и фамилия
- `age` - возраст
- `phone` - номер телефона

### 2. TimirbaevAutoCourier
Наследник TimirbaevCourier с дополнительными полями:
- `car_model` - марка автомобиля
- `car_number` - государственный номер

### 3. TimirbaevPark
Класс для управления коллекцией курьеров:
- Добавление/удаление курьеров
- Сохранение в XML файл
- Загрузка из XML файла

### 4. TimirbaevUtilities
Статический класс с полезными методами:
- Проверка ввода чисел
- Валидация данных

## Сборка и запуск

### Требования:
- **Windows 7/10/11**
- **.NET Framework 4.7.2**
- **Visual Studio 2019/2022** (или выше)

### Инструкция:
1. Откройте файл `Timirbaev_Erik_Lab_5.sln` в Visual Studio
2. Нажмите **Ctrl+Shift+B** для сборки проекта
3. Нажмите **F5** для запуска

## Работа с файлами
Данные сохраняются в **XML формате**:
```xml
<ArrayOfTimirbaevCourier>
  <TimirbaevCourier>
    <id>1</id>
    <firstname>Иван</firstname>
    <surname>Петров</surname>
    <age>25</age>
    <phone>9215551212</phone>
  </TimirbaevCourier>
</ArrayOfTimirbaevCourier>
Особенности реализации
Сериализация XML - для сохранения данных
Наследование - AutoCourier наследуется от Courier
Полиморфизм - виртуальные методы CreateCourier/ShowCourier
Валидация ввода - проверка корректности данных
